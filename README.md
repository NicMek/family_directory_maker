# family_directory_maker
Quick folder maker for family tree projects. builds the following structure:
        
        [first_name_father][LAST_NAME_FATHER]_[first_name_mother][LAST_NAME_MOTHER]/
        ├── family/
        │   ├── documents/
        │   └── images/
        ├── [first_name_child1][LAST_NAME_FATHER]L/
        │   ├── documents/
        │   └── images/
        └── [first_name_child2][LAST_NAME_FATHER]/
            ├── documents/
            └── images/
